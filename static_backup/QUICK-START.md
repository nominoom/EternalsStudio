## 🎯 How to Use

### **Option 1: Open Locally**
1. Double-click `index.html` to open in your browser
2. OR open `sitemap.html` to see all pages at once

### **Option 2: Use Live Server (Recommended)**
1. Open VS Code
2. Install "Live Server" extension if you haven't
3. Right-click `index.html` → "Open with Live Server"
4. Website opens at `http://localhost:5500`

> **Note:** Using Live Server ensures all features work perfectly, especially the navbar component!
